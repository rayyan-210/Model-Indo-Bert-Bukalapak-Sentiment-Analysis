This project focuses on building a **sentiment analysis model** for Indonesian-language product reviews from **Bukalapak**, one of the largest e-commerce platforms in Indonesia. This project is intended for **academic purposes**, especially for students learning **Natural Language Processing (NLP)** and **deep learning-based text classification**.
